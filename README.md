📱 Android Applications Portfolio
=================================

This repository serves as a central collection for my Android development projects. Each folder contains a standalone application ranging from agricultural utility tools to network diagnostics.

🚀 Featured Projects
--------------------

🌾 Cane Tracker
---------------

**Domain:** Agriculture / Logistics

*   **Purpose:** Simplifies sugarcane loading by tracking worker trips and calculating net harvest weight.
    
*   **Key Logic:** Automates the subtraction of worker body weight from gross scale readings to determine actual cane weight loaded.
    
*   **Status:** \[Development / Alpha / Beta\]
    

📶 WiFi Inspector Pro
---------------------

**Domain:** Utility / Networking

*   **Purpose:** Professional tool for analyzing network strength, identifying connected devices, and optimizing WiFi performance.
    
*   **Key Features:** Real-time signal monitoring, IP discovery, and security auditing.
    
*   **Status:** \[Development / Alpha / Beta\]
    

🛠 Tech Stack
-------------

*   **Language:** Kotlin / Java
    
*   **UI:** Jetpack Compose / XML Layouts
    
*   **Architecture:** MVVM (Model-View-ViewModel)
    
*   **Local Storage:** Room Database / SQLite
